About This Test Suite

This test suite is designed to validate the html2md converter against various challenging scenarios:

  • Complex Layouts: Multi-column layouts, flexbox, grid, and nested structures
  • Code Examples: Syntax highlighting, multiple programming languages, and inline code
  • Edge Cases: Malformed HTML, special characters, and unusual nesting
  • Modern Features: HTML5 elements, web components, and semantic markup
  • Rich Content: Tables, lists, multimedia, and interactive elements

Running the Tests

To test the html2md converter with these pages:

# Start the test server
$ python tests/html2md_server/server.py

# In another terminal, run html2md on the test pages
$ m1f-html2md \
    --source-dir http://localhost:8080/page/ \
    --destination-dir ./tests/html2md_output/ \
    --verbose

# Or test specific selectors
$ m1f-html2md \
    --source-dir http://localhost:8080/page/ \
    --destination-dir ./tests/html2md_output/ \
    --outermost-selector "article" \
    --ignore-selectors "nav" ".sidebar" "footer"

Contributing

To add new test cases:

  1. Create a new HTML file in tests/html2md_server/test_pages/
  2. Add an entry to TEST_PAGES in server.py
  3. Include challenging HTML structures that test specific conversion scenarios
  4. Document what the test page is designed to validate
Tip: Use the browser's developer tools to inspect the HTML structure and CSS styles of each test page.
